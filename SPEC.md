# SPEC.md — Файно натурально

## Продукт

Інтернет-магазин натуральних продуктів: спеції, приправи, чаї.
Клієнт може переглядати каталог, додавати товари в кошик і оформлювати замовлення без реєстрації.
Адміністратор керує товарами та замовленнями через адмін-панель.

---

## Стек

| Шар | Технологія |
|-----|-----------|
| Backend | ASP.NET Core 10 Web API |
| ORM | Entity Framework Core 10 + Npgsql |
| База даних | PostgreSQL 16 |
| Frontend | Angular 22 (standalone, signals) |
| Стилі | Tailwind CSS |
| Auth | JWT (access + refresh tokens) |
| Контейнери | Docker Compose |
| CI/CD | GitHub Actions |

---

## Фічі (пріоритет)

1. **Каталог товарів** — перегляд, фільтри, пошук
2. **Сторінка товару** — деталі, фото, "додати в кошик"
3. **Кошик** — сесійний, без авторизації
4. **Чекаут та замовлення** — форма, підтвердження
5. **Авторизація** — реєстрація, вхід, профіль
6. **Адмін панель** — CRUD товарів та замовлень

---

## Нефункціональні вимоги

- API відповідь < 200ms для GET запитів
- Mobile-first дизайн
- SEO: SSR або pre-rendering для сторінок каталогу та товару
- Всі форми — валідація на frontend і backend
- HTTPS в production
- Логування помилок (Serilog)
